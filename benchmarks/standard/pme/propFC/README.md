# Standard benchmark — PME — propFC

This benchmark defines the **standard dataset-selection** workflow for the **propFC** case using **PME**.

## Files

- `case.json` — main configuration for the dimensionality-reduction run
- `backmapping.json` — example backmapping configuration

## Dataset

The benchmark definition is present in the repository, but public dataset packaging for `propFC` is not yet aligned with the `glider` and `airfoil` dataset structure under `databases/`.

## Notes

Use this case as a repository benchmark definition, but verify local dataset availability before running it.
