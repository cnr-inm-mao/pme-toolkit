# Databases

This directory contains **dataset metadata and repository-level notes** for the benchmark problems.

Large benchmark datasets are generally **not stored directly in Git**. Instead, the repository keeps metadata describing where the datasets live and how they are used by the benchmark workflows.

## Currently documented datasets

- `glider`
- `airfoil`

## Notes on `propFC`

Benchmark definitions for `propFC` are present under `benchmarks/`, but in this repository snapshot the dataset side is not yet packaged and documented in the same way as `glider` and `airfoil`.
