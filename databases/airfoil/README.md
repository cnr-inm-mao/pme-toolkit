# Airfoil dataset metadata

This folder contains the repository metadata for the **airfoil benchmark dataset**.

## Purpose

The metadata stored here documents the dataset used by the airfoil benchmark cases defined under `benchmarks/`.

## Typical contents

In the current repository snapshot this folder contains `dataset.json`, which records the dataset-level information required by the repository workflow.

## Notes

Dataset availability and local paths still depend on the execution environment in which the benchmark is run.
