# Benchmarks

The repository contains benchmark definitions for the following cases:

- `glider`
- `airfoil`
- `propFC`

These are organized by selection strategy and method family under `benchmarks/`.

## Current maturity by case

- **glider**: best aligned benchmark case in the repository;
- **airfoil**: benchmark and dataset metadata present;
- **propFC**: benchmark definitions present, but dataset packaging still not aligned at the same level.
